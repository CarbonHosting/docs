# Why we rely on GitHub

GitHub is a platform which integrates with the Git CLI to host code repositories for large projects.

We integrate with GitHub to retrieve your files you have upload there and display them as a Toggled project.

## Create a GitHub account

Head to [https://github.com/signup](https://github.com/signup) and complete the form.
